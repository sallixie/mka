APA SIH ITU DIVISI GAME DEVELOPER?

Divisi Game Developer merupakan divisi yang bertugas menciptakan video game. Seorang game developer sehari-hari menggeluti pekerjaan yang berkaitan dengan pembuatan game. Area pekerjaannya mencakup pembuatan audio, desain grafis permainannya, seni visual, juga produksi aplikasinya. Seorang game developer harus memiliki beberapa skill seperti kemampuan IT yang kuat, kreativitas yang tinggi, kecintaan pada game, kemampuan problem solving, dan manajemen waktu. Prospek kerja game developer cukup menjanjikan karena terdapat potensi industri game di Indonesia yang besar. Secara umum, terdapat beberapa posisi yang dapat dipilih sebagai game developer seperti game designer, game artist, animator, game programmer, sound engineer, dan game taster.

Yuk, daftarkan dirimu segera menjadi bagian dari divisi Game Developer PT. Media Kreasi Abadi dan dapatkan pengalaman yang tentunya dapat menambah kompetensi kalian dalam membuat sebuah game yang berguna untuk menunjang karir kedepannya.

Kriteria divisi Game Developer

  • Mahasiswa perguruan tinggi aktif yang menempuh semester 5, 6 atau 7;
  • Berasal dari Jurusan Teknologi Informasi atau Rekayasa Perangkat Lunak atau DKV atau Matematika atau Fisika program studi yang setara;
  • Mampu berkomunikasi dengan baik dan bekerja dalam kelompok;
  • Memiliki inisiatif tinggi, kemauan belajar tinggi, siap berkolaborasi; dan
  • Bersedia ditempatkan pelanggan yang lagi butuh solusi.
