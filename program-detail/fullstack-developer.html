APA SIH ITU DIVISI FULLSTACK DEVELOPER?

Divisi Fullstack Developer merupakan divisi yang bertugas membentuk dan menganalisis front end dan back end web atau aplikasi. Seorang fullstack developer dituntut untuk dapat menguasai kedua bidang tersebut. Secara garis besar seseorang yang menjadi full stack developer wajib menguasai javascript, php, database, java (backend) dan juga dapat mengkoordinasikan desain ke dalam pemrograman seperti HTML, CSS, XML(front end). Tugas dan tanggung jawab utama seorang fullstack developer adalah merancang mekanisme interaksi pengguna dengan web atau aplikasi, mengelola server dan database agar web berfungsi dengan baik, serta melakukan coding untuk platform mobile.

Yuk, daftarkan dirimu segera menjadi bagian dari divisi Fullstack Developer PT. Media Kreasi Abadi dan dapatkan pengalaman yang tentunya dapat menambah kompetensi kalian dalam membuat sebuah aplikasi yang berguna untuk menunjang karir kedepannya.

Kriteria divisi Fullstack Developer

  • Mahasiswa perguruan tinggi aktif yang menempuh semester 5, 6 atau 7;
  • Berasal dari Jurusan Teknologi Informasi atau Rekayasa Perangkat Lunak atau DKV atau Matematika atau Fisika program studi yang setara;
  • Mampu berkomunikasi dengan baik dan bekerja dalam kelompok;
  • Memiliki inisiatif tinggi, kemauan belajar tinggi, siap berkolaborasi; dan
  • Bersedia ditempatkan di pelanggan yang butuh solusi atau kantor.
